Usar git con github
# Primero es necesario tener git instalado y una cuenta en github

# Crear configuraciones globales para git en la computadora

# Identificarse git config --global user.name "Tu username"
git config --global user.name "Tu username" # Seleccionar vscode como el editor git config --global core.editor "code --wait" # Configurar el salto de linea para Linux git config --global core.autocrlf input # Ubicarnos en el directorio que deseamos usar como repositorio, ya sea seleccionando uno ya existente, creando uno nuevo o descargando uno de github con: git clone https://direcciondelrepositorio.com #
