Lista de programas que se instalan facilmente
# Con los repositorios de apt, flatpak y snap, los paquetes se actualizan automaticamente
Apt
# Para ver si el paquete esta disponible apt search NombredelPaquete # Para instalar el paquete sudo apt install NombredelPaquete # Para desinstalar un paquete sudo apt purge NombredelPaquete # Para actualizar los paquetes sudo apt update sudo apt upgrade # Elimina los archivos innecesarios sudo apt autoremove
Flatpak
# Instalar flatpak con sudo add-apt-repository ppa:flatpak/stable
sudo apt install flatpak
 # Agregar el remoto flathub en user, system o en los dos flatpak remote-add --user --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo
flatpak remote-add --system --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo # Si agregemos los dos instalar un paquete, nos preguntara si instalar para user o para system. Usar system si queremos que el paquete este disponible para todos los usuarios en la computadora; sino, usar user

# Para ver si el paquete esta disponible flatpak search NombredelPaquete # Para instalar el paquete flatpak install NombredelPaquete # Para desinstalar un paquete flatpak uninstall NombredelPaquete # Para actualizar los paquetes flatpak update
Snap
# Para ver si el paquete (programa) esta disponible en los repositorios de snap snap find NombredelPaquete # Para instalar el paquete sudo snap install NombredelPaquete # Para desinstalar un paquete sudo snap remove NombredelPaquete # Para actualizar un paquete sudo snap refresh
Descarga directa
# Si el paquete no esta disponible en ninguna de las opciones, se puede descargar el archivo .deb wget https://Direccion/delaPagina/NombredelPaquete.deb
sudo apt install ./NombredelPaquete.deb

# Se puede usar cualquiera de las alternativas, siendo mas recomendable descargar de repositorios y no del archivo .deb, porque este ultimo no se puede actualizar

# Los repositorios con los paquetes mas actualizados son preferibles, pero a veces presentan errores, la siguiente lista dice de donde descargar paquetes de manera que esten actualizados y estables
  • Git con apt
    • sudo add-apt-repository ppa:git-core/ppa
    sudo apt install git
  • Lutris con flatpak
    • flatpak install flathub net.lutris.Lutris
  • Spotify con flatpak
    • flatpak install Spotify
  • RetroArch con flatpak
    • flatpak install retroarch
  • Firefox por snap
    • sudo snap install firefox
  • Discord descargando el .deb de su pagina oficial (actualizaciones automaticas sin repositorio PPA)
  • Steam descargando el .deb de su pagina oficial (actualizaciones automaticas sin repositorio PPA)
  • Visual Studio Code descargando el .deb se su pagina oficial (actualizaciones automaticas sin repositorio PPA)
  • Lutris (Para LoL) descarnago el .deb de su repositorio en github (sin actualizaciones automaticas)
