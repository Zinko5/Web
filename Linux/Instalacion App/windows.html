Programas o juegos de Windows
Para instalar programas o juegos nativos de Windows en Linux, primero necesitamos Wine, tambien Lutris es util para ordenarlos y arrancarlos

Primero debemos descargar los archivos del programa Windows y despues seleccionar o crear el directorio donde queremos que este el programa en Linux (ej. "~/.local/share/NombrePrograma")

# Crear directorio mkdir ~/.local/share/NombrePrograma # Crear prefix WINEPREFIX="~/.local/share/NombrePrograma" wine wineboot # Abrir una terminal en la carpeta en donde están los archivos Windows del programa y poner WINEPREFIX="~/.local/share/NombrePrograma" wine NombreArchivo.exe # Instalar el programa usando el asistente de instalacion

# Ahora, el programa estará instalado en ~/.local/share/NombrePrograma, podemos ejecutarlo con wine ~/.local/share/NombrePrograma/...(BUSCAR_LA_UBICACION_DEL_EJECUTABLE).../... NombreArchivo.exe Para ejecutar desde Lutris

# Abrir Lutris y dar click en el simbolo de + ubicado en la esquina superior izquierda.
En "Nombre" colocamos el nombre que queramos que tenga el juego y en "Ejecutador" seleccionamos Wine

# En la pestaña "Opciones de juego", en "Ejecutable" le damos en examinar y buscamos dentro de ~/.local/share/NombrePrograma el archivo NombreArchivo.exe o NombreArchivo.lnk y lo seleccionamos.

Es posible que para que algunos juegos se ejecuten correctamente, se debe personalizar la configuracion en la seccion de Opciones Avanzadas de las demás pestañas
