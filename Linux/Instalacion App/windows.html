Programas o juegos de Windows
Para instalar programas o juegos nativos de Windows en Linux, primero necesitamos Wine, tambien Lutris es util para ordenarlos y arrancarlos

Primero debemos descargar los archivos del programa Windows y despues seleccionar o crear la carpeta donde queremos que este el programa en Linux (ej. "UBICACION_DE_LA_CARPETA_LINUX")

# Crear prefix WINEPREFIX="UBICACION_DE_LA_CARPETA_LINUX" wine wineboot # Abrir una terminal en la carpeta en donde están los archivos Windows del programa y poner WINEPREFIX="UBICACION_DE_LA_CARPETA_LINUX" wine NOMBRE_DEL_ARCHIVO.exe # Instalar el programa usando el asistente de instalacion

# Ahora, el programa estará instalado en UBICACION_DE_LA_CARPETA_LINUX, podemos ejecutarlo con wine UBICACION_DE_LA_CARPETA_LINUX/...(BUSCAR_LA_UBICACION_DEL_EJECUTABLE).../... NOMBRE_DEL_ARCHIVO.exe Para ejecutar desde Lutris

# Abrir Lutris y dar click en el simbolo de + ubicado en la esquina superior izquierda.
En "Nombre" colocamos el nombre que queramos que tenga el juego y en "Ejecutador" seleccionamos Wine

# En la pestaña "Opciones de juego", en "Ejecutable" le damos en examinar y buscamos dentro de UBICACION_DE_LA_CARPETA_LINUX el archivo NOMBRE_DEL_ARCHIVO.exe o NOMBRE_DEL_ARCHIVO.lnk y lo seleccionamos.

Es posible que para que algunos juegos se ejecuten correctamente, se debe personalizar la configuracion en la seccion de Opciones Avanzadas de las demás pestañas
