Photoshop
La carpeta "Linux-PhotoshopCC-v22" estara en la ubicacion en la que usaremos la terminal (ej. "UBICACION_DE_LA_CARPETA")

# Clone the repo git clone https://github.com/YoungFellow-le/photoshop-22-linux.git
cd photoshop-22-linux # Run the main-menu file: ./main-menu.sh # In the main menu, you will see these options: [Choose options 1-6 or 7 to exit]:

# To install photoshop select option "1"

# Install Camera Raw (y/n): y" # Do not install Vdk3d (y/n): n" # You can always install Vdk3d it afterwards by selecting option "4":

# And to uninstall Vdk3d: "[Choose options 1-6 or 7 to exit]: 4 # To uninstall Photoshop: "[Choose options 1-6 or 7 to exit]: 2 # If you want to completely remove this installer, then delete the cloned folder after running the uninstaller.

# Dentro de /home/gabo/photoshop-22-linux/Ps-prefix/drive_c/Program Files/Adobe hay que colocar la carpeta de Photoshop que usaremos y cambiarle el nombre a "Adobe Photoshop 2021"

# Si no aparece un acceso directo al programa, podemos arrancalo con bash -c '/UBICACION_DE_LA_CARPETA/photoshop-22-linux/scripts/launcher.sh' # El codigo de arriba aparece tambien al finalizar la instalacion de Photoshop

# En mi caso, el codigo es bash -c '/home/gabo/photoshop-22-linux/scripts/launcher.sh'
