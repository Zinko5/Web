Mains
Cambiar mains cuando cambie el meta para usar siempre campeones fuertes.

Usar dos o maximo 3 champs fuertes, que no tengan demasiado alto pickrate o banrate, que no sean demasiado dificiles y que sean para ganar.

Los mains deben cumplir distintas funciones para ayudar en distintas composiciones.

Comprender la build para cada main.

Aprender a farmear con cada main.

Aprender el rango de los basicos de cada main para pokear.

Aprender los combos de cada main para tradear.

Tener en cuenta el cooldown de tus habilidades.

Hay que dominar todas las skillshoot y habilidades del campeon.
